코딩 애플 리액트(2)

리액트 설치와 개발환경 세팅

원하는 dir를 우클릭해서 터미널을 열고
npx create-react-app APPNAME
입력해서 프로젝트 시작

허가되지 않은 스크립트 에러뜨면
- powershell 관리자 권한으로 open
- Set-ExecutionPolicy Unrestricted 입력
그래도 안될 때 try
- npm install -g npm@latest


에디터(VScode) 열고 생성한 APP 폴더 열면 뭐가 여러 개 있는데
여기서 src/App.js에 코딩하면 된다

미리보기
- npm start(npm -v도 동일)
** 오류 발생
- 환경 변수 설정에서 시스템변수 nodejs를 bin까지 들어가는 경로로 들어갔더니 경로 이름이 틀려서 모듈을 찾지 못하는 오류가 발생했음
- 다시 설정하고 컴퓨터 다시 시작하니까 잘 됐음

node_modules : 프로젝트 구동에 필요한 라이브러리 코드 보관함(일반적으로 그냥 내비두면 된다)
public: static 파일 모아놓는 곳(아직 건드릴 필요 X)
src: 소스코드 보관함_코드 짜는 곳
App.js - main page

js 파일에 html 짜도 되네 ?!
그 이유: App.js에 있던 html을 index.html로 집어넣어주는 index.js 파일이 있기 때문

package.json: 프로젝트 정보/라이브러리 명까지 포함
