리액트에서 레이아웃 만들 때 쓰는 JSX 문법 3개

div 하나 남기고 다 지웠음

리액트라고 다를 거 없음
- html css 써서 div 박스 만들고 레이아웃 짜서 디자인 하면 된다

JS 파일에서 html을 쓸 수 있다
=> 실은 html이 아니라 JSX다.
JSX: .js 파일에서 쓰는 html 대용품(부가기능 or 언어라고 생각)


JSX문법1. 요소(div/p/h1..)에 class 넣을 때는 className으로

참고. css 파일 쓰려면 상단에서 import 'css경로';

JSX문법2. 변수 넣을 때는 {중괄호}

let 변수 = '~';
ex. id, href, src 전부 다 = {변수}로 사용 가능
=> 용어: 데이터 바인딩

JSX문법3. style 넣을 때는 style = { {스타일명: '값'} }
ex.
<h4 style = { {color: 'red'} }> 블라블라블라 </h4>
이어서 다른 스타일을 추가하고 싶으면
<h4 style = { {color: 'red', fontSize: '16px'} }> 블라블라블라 </h4>
css에서 폰트 사이즈는 font-size이지만, JS에서는 fontSize로 적는다
JS -는 진짜 뺄셈이기 때문이다.
